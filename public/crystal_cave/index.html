Using Unity's animation statemachine and retargeted animations from online, I am able to create a logical animation tree that allows bipeds to walk, stop, sit and stand in natural order. Using the LegsAnimator Unity asset to override the feet and hips so that they align with the surface and give a natural posture.
Using Blender I created a 6 legged testing rig for the LegsAnimator asset to fully animate the legs like a spider, allowing them to lift, move and land in my desired speed, location and frequency. This required some slight tweaking of the code in order to allow for unlimited input values for speed and step duration.
Fully 3D modeled base mesh for creation spirits Yee and Byamee who are the bipeds wandering the lands. Based on hand drawn sketches which are based on the Indigenous Australian creation spirit Baiame. I 3D modeled and rigged these in blender for animation retargeting in Unity.
In order to have the creation spirits walking around I downloaded several physical root animations from online and used 3dsMax's CAT system to retarget them onto my rigs. Then using UmotionPro to tweak the posture of the characters. Enjoy this video I made for John to show him how to use CAT.
