Romanesque

Unreal Engine 5

2024-10

Romanesque is my first deep dive into game technology and a full stack of software required to make highly detailed 3D sculptures, for a virtual environment. This is my first seriously detailed project and took a lot of time and effort. I wanted this to be proof that I can do anything and truly impress myself with my own art, and I think I have done that. In this post I will go over the tools and techniques I used, I'll show some in progess versions and at the end I'll go over how I can use these technologies both in Unreal Engine and Unity. This project was done following a tutorial by FastTrack Tutorials

This project starts in Blender, using reference images. The first thing I've learnt here is that the easiest way for me to make a new model from scratch is to start with bezier curves. In the 2D side view, using the curves to make the profile of an object firstly, allowing for smooth edges and corner with ease. This makes the process really simple, then I can extrude the profile into the 3rd dimension. If its a cuboid, then shearing the profile into a corner is a helpful trick for making a corner while conserving all of the details in the profile. Models come out looking great!
